GUIDE TO RUN THE TO-DO-LIST-APP.

1. Open the "index.html" file with a browser.
2. There is a field at the top to type in your TASK.
3. Click the ADD BUTTON to add the task you just added.
4. Click the BLUE BUTTONN with the edit icon to edit your task for the sake of any changes.
5. Tick the "cirle" shape close to the task created to mark it as completed.
6. Click the RED BUTTON with the delete icon to remove task once completed.
